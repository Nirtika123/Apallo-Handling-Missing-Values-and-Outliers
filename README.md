# Lab 3: Handling Missing Values and Outliers

## Aim
To identify missing values, calculate missing-value percentages, perform suitable imputation, detect unrealistic observations, identify outliers using the IQR method, validate BMI values, and prepare a cleaned patient health-screening dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Tasks Completed
- Loaded and inspected the dataset
- Identified missing values
- Imputed missing numerical and categorical values
- Detected unrealistic values
- Validated BMI calculations
- Detected outliers using the IQR method
- Treated invalid observations
- Generated the final cleaned dataset
